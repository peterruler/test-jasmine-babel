# test-jasmine-babel

> Test project which use Babel.js as ES6 compiler to write unit tests

## Setup

```
npm install
```

## Test

Before you should have global installation of `jasmine` package.

```
npm test
```

## Support 

Current support version:

 - `Babel ^6.3.17`

Previously support:

 - `Babel ^5.8.29`
